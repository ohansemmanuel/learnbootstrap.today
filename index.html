Another book?

We seriously don't need one

...When I got my first job, I realized something no one told me

Apart from being hired to do a goob job, you are expected to ship products fast!

Why do you think Bootstrap has the highest number of github stars for a frontend project?

Rapid prototyping is a big thing, and you need quick tools to get the job done faster :-)

Beautiful apps, shipped fast, with more productivity What more could you ask for?

Do I get all this sweetness for free?

They say the best things are free. Maybe not this.

Bootstrap itself is a relatively easy technology to pick up. The "friends" of Bootstrap are arguably not.

For efficiency, you should use Bootstrap with other interesting technologies such as SASS, Webpack and ReactJS

Creating your own build process using Gulp / Webpack is almost a must have these days :) Ask any experienced developer.

Also, a lot of people make a decent living off of creating and selling professional Bootstrap themes eg. admin dashboard themes

It is really difficult to find a single resource that teaches you all of this in plain, approachable language

Well, that's what this book does!

Whether you're a beginning front end developer looking to ace your first job, or a seasoned developer looking to be more productive on the frontend, this book's got you covered.
